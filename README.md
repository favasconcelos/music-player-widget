## Music Plater Widget
Inspired on [Music Player Widget] by [Rovane Durso]
Checkout at: [Online Version]

---

### Install

Install dependecies

```sh
bower install
```

To run:

```sh
npm install
gulp
```
open browser at: [http://localhost:8080/](http://localhost:8080/)
